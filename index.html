Modelagem com Grafos — Grafo Direcionado

Representação da MEF como um grafo direcionado G = (V, A), onde V = {E1, E2, E3, E4} são os vértices (estados) e A = {(E1,E2), (E2,E3), (E3,E4), (E4,E1)} são as arestas (transições). O grafo é cíclico e fortemente conexo.

E1 — Via A verde (26s)
E2 — Via A amarelo (4s)
E3 — Via B verde (26s)
E4 — Via B amarelo (4s)

Modelagem com Teoria dos Conjuntos

O sistema é modelado usando conjuntos para representar veículos, pedestres e os estados do sistema.

Definição dos Conjuntos

V (veículos)= { v₁, v₂, v₃, ..., vₙ }
VA (veículos Via A)⊂ V
VB (veículos Via B)⊂ V
P (pedestres)= { p₁, p₂, p₃, ..., pₙ }
E (estados)= { E1, E2, E3, E4 }

Operações entre Conjuntos

VA ∩ VB = ∅ (interseção vazia — veículo não pode estar nas duas vias)
V = VA ∪ VB (total de veículos é a união)

Conjuntos por Variável Binária (estados onde a variável = 1)

A (Via A verde)= { E1 }
B (Via B verde)= { E3 }
PA (Ped A livre)= { E3, E4 }
PB (Ped B livre)= { E1, E2 }
YA (Via A amarelo)= { E2 }
YB (Via B amarelo)= { E4 }

Operações Derivadas

A ∩ B= ∅ (comprova ¬(A ∧ B))
A ∪ B= { E1, E3 }
PA ∩ PB= ∅ (pedestres A e B nunca livres ao mesmo tempo)
PA ∪ PB= { E1, E2, E3, E4 } (sempre há pedestre cruzando)
YA ∩ YB= ∅ (nunca duas vias em amarelo)
Complemento de A= { E2, E3, E4 }
Complemento de B= { E1, E2, E4 }
